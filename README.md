## Trinity Agent v1 to v6

This is a Mixture of Experts agent consists of:

- Trend Agent: Transformer framework aiming to profit when the ticker is surging
- MeanRev Agent: Mean Reversion Strategy targetting chopping
- Crisis Agent: exclusively for saving the agent during bear market. This agent is extremely effective, gaining an alpha of 20% on GOOG from 2024 to 2025.
